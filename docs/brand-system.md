# AdvantaCode Brand System

## Brand Direction
- Positioning: Systems Engineer / Automation Architect
- Tone: Confident, intentional, technical, modern
- Vibe: Clean edge-first infrastructure

---

## Color System

### Base Background
- Zinc 950

### Primary Accent
- Amber 500
- Hover: Amber 400
- Active: Amber 600

### Neutral Scale
- Zinc 100–900

### Semantic
- Success: Emerald 500
- Warning: Amber 500
- Error: Red 500

---

## Typography

### Headings
- Font: Geist
- Weights: 600, 700

### Body
- Font: Inter
- Weights: 400, 500

### Code
- Font: JetBrains Mono
- Usage: Snippets, CLI, automation examples

---

## Layout Philosophy
- Generous whitespace
- Clear hierarchy
- Strong CTA contrast
- Minimal animations
- Motion only for clarity

---

## UI Feel
- Rounded corners: lg
- Shadow: subtle
- Border usage: light, mostly for separation
- Buttons: solid primary, ghost secondary

---

## Brand Keywords
- Automation
- Infrastructure
- Systems
- Edge-first
- Leverage
- Optimization
- Engineering clarity

---

## Emotional Target
When someone lands on AdvantaCode, they should feel:
- This person understands systems.
- This feels modern.
- This feels scalable.
- This feels intentional.
